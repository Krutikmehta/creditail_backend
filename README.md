# creditail_backend
- Nodejs, MongoDB
- invoice schema - refer [models/invoice.js](models/invoice.js)
- Index on db : billNo
- host - creditail-backend.vercel.app
- endpoints
    - POST - /api/v1/invoice
        - add a invoice
        - body : invoice schema
    - POST - /api/v1/invoices 
        - add multiple invoices
        - body : [invoice schema]      
    - GET - /api/v1/invoices      
        - get all invoices  
    - POST - /api/v1/update_invoice 
        - update a invoice
        - body : { billNo: String, pendingAmount: Number }
